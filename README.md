# vue3-aide

- vue3 + vite + ts + vue-router + pinia + vant + axios
- npm install
- npm run dev // 启动项目
- npm run build // 打包项目
- npm run preview
- npm run deploy // 部署项目

## 项目集成

- npm install gh-pages --save-dev // 部署项目到 github pages

```
    "scripts": {
    "predeploy": "vite build",
    "deploy": "gh-pages -d dist"
    }
```
