# vue3-aide
- vue3 + vite + ts + vue-router + pinia + vant + axios
- pnpm install      安装依赖
- pnpm run dev      启动项目
- pnpm run build    打包项目

# 发布项目
- npm run build
- 提交到远程
- 自动更新页面

## 项目集成-后续更新

    "scripts": {
    "predeploy": "vite build",
